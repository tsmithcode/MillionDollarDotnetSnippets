# Product Surface

MillionDollarDotnetSnippets is presented as a product for high-value delivery work, not as a historical sequence of phases.

## Build Faster

Use these helpers when you need to reduce setup drag and move from blank project to working capability faster.

- reflection and runtime construction
- object mapping
- LINQ shaping
- JSON and dictionary transformations
- dependency wiring

## Ship Safer

Use these helpers when the project is already moving and you need stronger runtime behavior.

- retry logic
- exception handling
- timing and logging
- validation guards
- config and environment reads
- lightweight caching

## Automate More

Use these helpers when you want repeated engineering steps to become tooling.

- expression trees
- dynamic filters and setters
- CLI helpers
- code-generation-style template utilities
- file and stream workflows

## Integrate Messy Systems

Use these helpers when the real work is at the boundaries:

- parsing ugly inputs
- normalizing strings and enums
- stabilizing environment-specific behavior
- handling repeated transformations between systems

## Internal note

The phase-based source layout is still useful as an implementation taxonomy.
It should remain subordinate to the product story, not the other way around.
