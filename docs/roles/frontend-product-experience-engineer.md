# Frontend Product Experience Engineer

## Mission

Build the public surface as a guided product experience, not a static marketing shell.

## Owns

- website interaction model
- wizard flow implementation
- zero-knowledge onboarding path
- responsive and accessible interaction quality

## First 30 days

- define the story-framework wizard flow
- implement the first guided path with clear progress states
- make desktop and mobile behavior feel equally intentional

## Scorecard

- a first-time user can reach the right proof path quickly
- the public surface feels like product onboarding
- interaction patterns stay clean, fast, and accessible
