# Brand System

Million Dollar Dot Net Snippets should feel premium, sharp, and commercially valuable without becoming visually noisy.

## Visual thesis

Warm executive-grade palette over dark foundations.
The look should feel like a high-margin consulting artifact, not a generic developer template.

## Core rules

- typography should be bold, restrained, and easy to scan
- brand assets should use short phrases, not paragraphs embedded in SVGs
- contrast must stay comfortably readable
- decorative polish must never hide the product promise
- every visual should reinforce the same buyer: high-value .NET consultant

## Asset rules

- all SVGs need `title` and `desc`
- hero assets should communicate one idea fast
- copy embedded in visuals should stay short and high-contrast
- visuals must still make sense when read by assistive technology

## README rules

- first screen must show product, persona, and next step
- navigation links should stay short and meaningful
- sections should read cleanly from top to bottom in linear order

## Ownership

- Design / Brand Systems Lead owns the system
- Accessibility & UX Quality Lead can block changes that reduce clarity
- Product / Positioning Lead ensures the visuals support the market story
