# Examples

This folder contains runnable examples that show how to adopt the repo quickly.

## Available examples

- [ConsultantQuickstart](./ConsultantQuickstart)

Use the quickstart when you want to prove the library, inspect real usage, and copy a working pattern into your own project.

The quickstart currently demonstrates:

- file ingestion
- HTTP-backed ingestion
- config-driven rules
- validation failures with clear messages
- record-by-record diagnostic trace output
